招聘与入职管理办法

文档编号：HR-REC-002

版本号：V1.1

生效日期：2026-02-01

适用范围：招聘团队、用人部门、行政与信息支持部门

一、招聘原则

公司坚持公开、公平、择优录用的招聘原则，根据岗位编制、业务发展计划及预算安排开展招聘工作。

二、招聘流程

  1. 用人部门提交招聘需求申请。
  2. 人力资源部审核编制、预算及岗位说明书。
  3. 发布招聘信息并筛选简历。
  4. 组织笔试、面试与背景调查。
  5. 发放录用通知并办理入职手续。

三、面试评价维度

评价维度 说明 建议权重
专业能力 岗位相关知识、项目经验、问题解决能力 40%
沟通协作 表达清晰度、跨团队配合意识、反馈能力 20%
价值观匹配 职业操守、责任意识、组织认同感 20%
发展潜力 学习能力、适应变化能力、成长预期 20%

四、录用审批

候选人拟录用前，须完成以下审批：

  • 直属主管确认录用意见
  • 部门负责人确认编制与薪级建议
  • HR确认薪酬区间与入职时间
  • 关键岗位需完成背景调查

五、入职材料清单

材料名称 是否必需 备注
身份证复印件 是 需核验原件
学历学位证明 是 海外学历需补充认证材料
离职证明 是 应届生可不提供
银行卡信息 是 用于薪资发放
一寸照片 否 用于工牌和系统资料

六、试用期管理

新员工入职后进入试用期，试用期通常为 3 个月，特殊岗位可根据劳动合同约定执行。试用期内由直属主管进行工作辅导，并在试用期结束前完成转正评估。

七、附则

本办法由人力资源部负责解释与修订。涉及法律法规变化时，按最新规定同步调整。
